אפליקציה להדמיה מנטלית והבנת הנקרא

כלי חדשני ומתקדם המיועד לשיפור הבנת הנקרא באמצעות טכניקות הדמיה מנטלית. האפליקציה משלבת בינה מלאכותית ליצירת תרגילים מותאמים אישית, דפי עבודה אינטראקטיביים, וכלים פדגוגיים מתקדמים למורים ולתלמידים.

  • ✓ יצירת טקסטים מותאמים אישית באמצעות בינה מלאכותית
  • ✓ תרגילי הבנת הנקרא עם שאלות חשיבה מגוונות
  • ✓ זיהוי אווירה ורגשות בטקסט
  • ✓ איורים ונרציה קולית לתמיכה בלמידה רב-חושית
  • ✓ ייצוא לWord ו-HTML לשימוש נוח בכיתה

⚠️ הערה חשובה לפני השימוש

האפליקציה דורשת מפתח API של Google Gemini לעבודה תקינה.

אם אתם מקבלים שגיאה שאין מפתח, עקבו אחר השלבים הבאים (הוראות למנהל המאגר):

  1. קבלו מפתח API חינמי מ-Google AI Studio
  2. הוסיפו את המפתח ב-GitHub Secrets של המאגר (שם: API_KEY)
  3. הריצו מחדש את ה-GitHub Actions Workflow
  4. המתינו 2-3 דקות לסיום הבנייה

להוראות מפורטות, ראו את הקובץ API_KEY_FIX.md במאגר.
